# Mintlify Clone

This project is a desktop-first documentation-style website inspired by the Mintlify website. The goal was to recreate the structure, layout, and design of the original Mintlify documentation site using only HTML and CSS.

## Features

### Sections Recreated
1. **Top Navigation Bar**
   - Includes the logo, navigation links, and primary call-to-action buttons.
2. **Hero Section**
   - Main headline, short description, email input with a CTA button, and a large background illustration.
3. **Documentation Preview Section**
   - Sidebar-style navigation and main content cards.
4. **Trusted By / Logos**
   - A row of company logos showcasing partnerships.
5. **Feature Highlights**
   - Two-column sections with text and visuals.
6. **Intelligent Assistant / UI Preview**
   - Large UI mockup with a description.
7. **Enterprise Features Section**
   - Title, short intro, and feature blocks (e.g., security, compliance).
8. **Case Studies / Customer Stories**
   - Card-based layout with images and descriptions.
9. **Final Call-To-Action**
   - Strong heading and CTA buttons.
10. **Footer**
    - Multi-column links and company/legal information.

### Fonts and Colors Used
- **Fonts**: Sans-serif fonts were used throughout the project to maintain simplicity and readability.
- **Colors**:
  - Primary: `#007a43`
  - Secondary: Various shades of black, white, and gray for text and backgrounds.
  - Accent: `#22c55e` for success indicators and highlights.

### Assets
- Images and icons were sourced from the provided brand assets or recreated to match the original design.

### Constraints Followed
- No JavaScript, TailwindCSS, or animations were used.
- The design is desktop-only and focuses on structural accuracy.
